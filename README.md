# 摸鱼背单词

一款专为开发者设计的隐蔽背单词工具，让你在写代码的同时轻松学习英语单词。

## ✨ 功能特点

- 🎭 **控制台伪装模式**：单词伪装成调试日志，完美融入开发环境
- 🧠 **艾宾浩斯遗忘曲线**：智能推荐需要复习的单词
- ⌨️ **快捷键操作**：Alt+Q/W/E 快速切换，无需鼠标
- 🛡️ **老板键**：一键隐藏所有学习内容，切换为正常调试信息
- 📚 **多词库支持**：CET4/6、考研核心词汇，共 585+ 单词
- ❤️ **生词本**：收藏难词，重点复习
- 📊 **学习统计**：掌握进度一目了然

## 🚀 快速开始

1. 安装插件后自动启动
2. 查看 VS Code 输出面板（Debug Console）或底部状态栏
3. 使用快捷键开始背单词

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+←` | 上一个单词 |
| `Alt+→` | 下一个单词 |
| `Ctrl+Alt+E` | 老板键（隐藏/恢复） |
| `Alt+↓` | 显示单词详情 |

## 📖 使用说明

### 控制台伪装模式（推荐）
单词会以调试日志的形式显示在输出面板中：

```
[10:30:45] [INFO] Module "abandon" loaded successfully
[10:31:12] [DEBUG] Variable "/əˈbændən/" initialized
[10:32:08] [CONFIG] v. 放弃，抛弃
```

外人看来只是普通的调试日志，实际上你在背单词！

### 智能推荐
系统会优先推荐：
1. 需要复习的单词（根据艾宾浩斯遗忘曲线）
2. 未学习过的新单词
3. 随机复习已掌握的单词

### 单词详情
点击状态栏单词或按 `Alt+D` 打开详情面板，包含：
- 单词音标和释义
- 例句及翻译
- 学习状态（新词/生词本/已掌握）
- 复习次数和下次复习时间
- 快捷操作按钮（上一个/下一个/已掌握/生词本）

## 🎮 高级功能

### 切换显示模式
`Ctrl+Shift+P` → `摸鱼背单词: 切换显示模式`

- **仅控制台**：单词显示在输出面板（最隐蔽）
- **仅状态栏**：单词显示在底部状态栏
- **状态栏+控制台**：双显示模式
- **完全隐藏**：暂停学习

### 查看生词本
`Ctrl+Shift+P` → `摸鱼背单词: 查看生词本`

浏览所有收藏的单词，点击可快速跳转。

### 学习统计
`Ctrl+Shift+P` → `摸鱼背单词: 学习统计`

查看学习进度：总词数、已掌握、生词本、待复习数量。

### 开始复习
`Ctrl+Shift+P` → `摸鱼背单词: 开始复习`

优先复习所有已到复习时间的单词。
